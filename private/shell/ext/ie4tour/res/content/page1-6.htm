Protect Your Computer with Security Zones
Internet Explorer 4.0 helps protect your computer by dividing Web sites into areas of trust, called zones. You can assign specific Web sites to the right zone with the appropriate security option settings. For restricted sites, your security settings are high, requiring approval every time you open a file or run a program from the Internet. For trusted sites or your secure local intranet (such as your company's internal network), your security settings can safely be set low.

Security Level


         
There are four different zones to choose from. For the Trusted Sites, Restricted Sites, and Local Intranet zones, click Add Sites to add the addresses for specific Web sites to that zone. Then, when you browse a site assigned to the zone, the security options for that zone will be applied. Once you select a zone, you can choose which security level you want for it.
