Terminal Services

Remote Administration Mode
Terminal Services also enables system administrators to remotely log on to and manage an entire Windows 2000 system from virtually anywhere in the world.

Application Server Mode
Terminal Services provides access to Windows-based applications by running the applications entirely on the server and sending only the display output to the client. Terminal Services facilitates centralized administration of applications; it is an efficient and cost effective way to manage older systems.

  • You must install Terminal Services Licensing to run in application server mode.

Caution
Installing Terminal Services affects the functionality of other Windows components. Microsoft recommends that you read the Deployment Guide before you install Terminal Services.

To set up Terminal Services, use the Windows Components wizard. After you finish with the wizard, your server will restart and the Configure Your Server screen will appear. Return to this screen to start using Terminal Services.
