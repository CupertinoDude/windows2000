Print Server

The Add Printer wizard helps you set up a local or network printer and publish to Active Directory. Run the wizard for each printer and platform you want to set up. After you run the wizard, network users can begin using the printer immediately. You can also administer the print server from any computer on the network by using the Printers folder or a Web browser.
