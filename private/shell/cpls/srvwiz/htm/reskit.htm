Windows 2000 Support Tools

The Windows 2000 Resource Kit contains valuable technical and performance information about Windows 2000, as well as many tools that enhance and extend the operating system.

To install the Windows 2000 Support Tools
1. Insert the Windows 2000 compact disc into your CD-ROM drive.
2. When the Autorun screen appears, click Browse This CD.
3. In Explorer, double-click the CD-ROM drive, double-click Support, and click Tools.
4. Double-click Setup and follow the instructions on your screen.
