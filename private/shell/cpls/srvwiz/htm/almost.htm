You're almost done. You need to set the static IP address, authorize DHCP, and restart the server. After these steps are completed, Active Directory, DNS, and DHCP will be configured.

To configure a static IP address
1. Right-click My Network Places and click Properties.
2. Right-click Local Area Network Connection and click Properties.
3. Click the TCP/IP component and click Properties.
4. Check Use the Following IP Address.
5. In the IP Address field, type:
10.10.1.1
6. In the Subnet Mask field, type:
255.0.0.0

To authorize the DHCP service
1. Open DHCP Manager.
2. Expand the DHCP server.
3. Right-click the server, point to All Tasks, and click Authorize.

When you have completed the steps above, click Ok. You need to reboot the machine to complete your server configuration.
