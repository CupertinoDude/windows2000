Routing

Routing services enable Windows 2000 to serve as a software-based router that provides multiprotocol LAN-to-LAN, LAN-to-WAN, virtual private network (VPN), and network address translation (NAT) routing services. Use the Routing and Remote Access Configuration wizard to set up routing.

To start the Routing and Remote Access Configuration wizard
1. Open Routing and Remote Access.
2. In the console tree, expand Routing and Remote Access, right-click your server, and click Install Routing and Remote Access.
