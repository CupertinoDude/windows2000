Database Server

Windows 2000 Server is a powerful database application server that integrates network operating systems and Internet services. With higher system scalability, support for large physical memories, and higher levels of system reliability and security, Windows 2000 Server is the ideal platform for database-intensive work, including
line-of-business, e-commerce and data-warehousing solutions.

Windows 2000 does not need further configuration. However, you need to install a database server to take advantage of Windows 2000 database capabilities.
